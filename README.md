# 📄 PDF → LaTeX 转换工具（via DeepSeek）

一个 Python 脚本，调用 DeepSeek 大模型，将中文数学建模论文 PDF 自动转换为结构清晰的 LaTeX 文档。

## ✨ 能做什么

- 提取 PDF 中的全部文字（支持多页）
- 自动生成符合中文排版习惯的 LaTeX 文件（`ctexart`）
- 将原文中的公式转为 LaTeX 数学环境
- 表格转为三线表（`booktabs`）
- 保留图片、参考文献的占位符
- 本地运行，你的 PDF 不会上传到任何地方（仅将提取的文字发送给 DeepSeek API）

## ⚠️ 已知限制（使用前必读）

本项目是一个辅助工具，**并非完美的一键转换方案**，存在以下已知限制：

- **表格处理**：PDF 中的表格会被提取为纯文字，**无法自动识别和重建表格结构**。生成的三线表可能需要手动核对和调整数据。
- **图片处理**：所有图片均转为 `\includegraphics` **占位符**，不会提取或生成实际图片文件。你需要手动从原 PDF 中导出图片并插入。
- **字体还原**：本工具不会分析原文档的字体，LaTeX 输出使用 `ctexart` 默认字体（宋体/仿宋/黑体）。**会与原文档字体有差异**，可自行在导言区修改字体设置。
- **格式偏差**：生成结果会力求结构完整，但细节排版（如段落间距、列表样式）可能与原文不完全一致。

> 💡 **建议**：将输出视为一份“粗排草稿”，在此基础上手动精修，远比从零写起高效得多。
